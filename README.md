# K-CORE-NCDC

K-CORE analysis portal repositories mirrored under `NCDCbioinformatics`.

Original source organization: https://github.com/K-CORE-NCDC

This repository acts as a landing page so the NCDC software profile can present both major tracks in one place:

- `CURE-NGS`: panel-based genomic data construction and harmonization framework
- `K-CORE`: clinico-omics analysis portal development

## K-CORE Portal Repositories

- `ncc-frontend`: React-based web client and visualization interface
- `ncc-backend`: Django-based backend services and portal logic
- `synthetic-data-set`: synthetic datasets used for portal validation and demonstration

## Notes

- The original `K-CORE-NCDC` repositories remain unchanged.
- The mirrored repositories under `NCDCbioinformatics` are maintained for unified public presentation.
- Portal website: https://www.cancerdata.re.kr/k-core/
